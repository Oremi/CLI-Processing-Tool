# CLI-Processing-Tool
A command-line text processing tool written in Go

Features

    Converts hexadecimal (hex) to decimal

    Converts binary (bin) to decimal

    Word case modifiers:
        (up)
        (low)
        (cap)
        (up, N)
        (low, N)
        (cap, N)

    Fix punctuation spacing

    Fix single quotes formatting

    Convert "a" to "an" before vowels or h

    Unit testing
